Open a new Krazimo mission.

Usage: /krazimo-open-mission <type>/<slug>
Example: /krazimo-open-mission feat/add-user-login

$ARGUMENTS

Use the `Skill` tool to invoke `krazimo-next-standards:krazimo-open-mission` with the argument above. The skill handles: creating the worktree, branch, mission folder, draft Issue, and draft PR.
